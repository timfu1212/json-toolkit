JSON 是現代開發者的日常語言，但在快速開發中，我們需要的不僅是格式化，而是能同時滿足**格式、壓縮、分析、轉換**的輕量化工具。

- **實用性極高**：針對開發者痛點設計，無廣告、無後端儲存，資料安全且快速。
- **技術架構純粹**：零依賴（Pure HTML/CSS/JS），程式碼透明且易於維護。
- **即時部屬**：單一檔案結構，上傳至 GitHub Pages 或靜態伺服器即可完美運行。
- **深度工程展現**：整合了語法解析演算法、複雜的 DOM 操作與極致的 UX 設計。

## 🎨 設計美學與亮點

預設導入專業 Monospace 字體，呈現最舒適的程式碼閱讀體驗，讓開發感覺更專業。

採用 **Sticky Header** 與雙側連動佈局，確保你在編輯與查看樹狀結構時，兩端狀態始終可見。

## ⚠️ 使用須知

### 檔案大小限制

本工具為純前端執行，為確保瀏覽器效能順暢：

- ✅ **建議**：處理 5MB 以下的 JSON 檔案
- ⚠️ **注意**：貼上大於 10MB 的 JSON 可能導致畫面暫時卡頓
- 💡 **提示**：如遇大型檔案，請耐心等待或分段處理

### CSV 轉換說明

將 JSON 轉為 CSV 表格時：

- ✅ **最佳格式**：陣列中包含相同結構的物件
  ```json
  [
    { "name": "Tim", "age": 30 },
    { "name": "Amy", "age": 25 }
  ]
  ```
